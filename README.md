# 本分支用于记录本人给自己的NEC7机器安装配置fedora 43 KDE
## 机器配置
主机：Lenovo NEC 7 <br>
CPU：Intel(R) Core(TM) i3-6100 (4) @ 3.70 GHz <br>
GPU：Intel HD Graphics 530 @ 1.05 GHz<br>
RAM：SK Hynix 8G<br>

## 目的
给NEC7安装fedora43 KDE，并使其支持RDP远程，用于作为网管机管理自己以及局域网内其他服务器

## 开始
### 下载fedora43 KDE Plasma
fedora官方下载地址（会重定向到镜像站）：https://download.fedoraproject.org/pub/fedora/linux/releases/43/KDE/x86_64/iso/Fedora-KDE-Desktop-Live-43-1.6.x86_64.iso <br>
fedora官方下载地址（BitTorrent）：https://fedoraproject.org/torrents/43/ <br>
### 刷写fedora43 KDE Plasma
本人使用Ventoy，直接把镜像放进去刷好Ventoy的移动盘里就行<br>
Ventoy项目地址：https://github.com/ventoy/Ventoy
